## Qué construyes

Un agente que:

1. Recibe un objetivo en lenguaje natural ("revisa mis correos no leídos y contesta los que sean preguntas simples").
2. Autentica con Gmail vía OAuth.
3. Lee los correos no leídos.
4. Decide cuáles son simples y cuáles requieren tu atención.
5. Redacta y envía respuestas a los simples.
6. Te resume al final qué hizo.

## Por qué importa

Es el patrón completo `tool use + agente + OAuth externo`. Lo que aprendas aquí se aplica a Slack, Notion, Linear, cualquier API con OAuth.

## Pasos (preview)

1. Configurar OAuth Gmail en Google Cloud Console.
2. Guardar tokens en `profiles` (encriptados).
3. Registrar tools: `listUnreadEmails`, `readEmail`, `sendReply`.
4. Armar grafo LangGraph: `fetch → classify → respond → report`.
5. Exponer la tarea como MCP server o como página.

## Entregable

Un agente que demuestras en Demo Day procesando tu propio inbox.
