## Setup inicial

1. [vercel.com/new](https://vercel.com/new) → Import tu repo de GitHub.
2. **Root Directory**: `web` (obligatorio porque es monorepo).
3. Framework: Next.js (autodetectado).
4. Build command: `yarn build` (autodetectado).
5. Install command: `yarn install` desde la raíz del monorepo.

## Variables de entorno

Settings → Environment Variables. Agrega todas las de tu `web/.env.local`. Marca scope:

- `Production` — para tu URL principal.
- `Preview` — para PRs.
- `Development` — para `vercel dev` local.

## Deploys automáticos

Cada `git push` a `main` deploya a producción. Cada PR genera un preview con URL única.

<Callout type="warning" title="Cambios en config.js requieren redeploy">
Como `config.js` se evalúa en build, cambiarlo en local no afecta producción hasta el siguiente push.
</Callout>
