# Firmware (ESP-Claw)

Código de referencia para el módulo de hardware (Sem 8–9 del Curso de Vibecoding Remotto).

> **Si estás en Sem 1–7, no necesitas tocar esta carpeta.**

## Qué hay aquí (Fase 6 del boilerplate)

```
firmware/
├── examples/
│   ├── 01-hello/         ← ESP-Claw básico, primer flash
│   ├── 02-mcp-bridge/    ← Conecta al /api/mcp de tu web app
│   └── 03-use-case/      ← Caso de uso preparado por el equipo docente
└── docs/                 ← Setup del kit, flasheo, troubleshooting
```

## Setup (Sem 8)

Documentación completa: [/docs/features/hardware-esp-claw](http://localhost:3000/docs/features/hardware-esp-claw) en tu instancia local de VibeFast.
