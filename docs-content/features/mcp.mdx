## Doble rol

- **Servidor MCP** en `/api/mcp` — expone el registry de tools. Cualquier cliente MCP (Claude Desktop, otros) puede usar tu producto como herramienta.
- **Cliente MCP** en `lib/mcp/client.js` — consume MCPs externos (Gmail, Notion, ESP-Claw).
