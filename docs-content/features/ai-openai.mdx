## Helpers principales

```js
import { streamChat } from "@/lib/openai/chat"
import { generateObject } from "@/lib/openai/structured"
```

- `streamChat(messages)` — chat streaming para UIs interactivas.
- `generateObject(schema, prompt)` — JSON tipado garantizado por el modelo.

Configuración de modelos y costo en `config.ai`.
