## Concepto: ESP-Claw

ESP-Claw es un framework de **chat para crear** que corre un agente de IA en el propio ESP32 (edge), con motor híbrido LLM + Lua. Defines comportamiento describiéndolo en lenguaje natural; el LLM decide y las reglas confirmadas se fijan como scripts Lua locales (ejecución determinista, offline).

Funciona como server y client MCP — enlaza directo con lo que hiciste en [Semana 5](/docs/tutoriales/semana-5-agentes-mcp).

## Pasos (preview)

1. Flashear ESP-Claw al ESP32.
2. Configurar WiFi del dispositivo.
3. Definir un comportamiento simple por chat.
4. Conectarlo a tu `/api/mcp` desde el ejemplo `firmware/examples/02-mcp-bridge/`.
